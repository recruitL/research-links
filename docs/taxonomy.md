# Taxonomy

## Topic

- `EOB`: Effective-One-Body theory and waveform models.
- `GSF`: gravitational self-force and radiation reaction.
- `BHPT`: black-hole perturbation theory.
- `QNM`: quasinormal mode methods and ringdown.
- `PM/EFT`: post-Minkowskian dynamics and effective field theory.
- `LISA`: waveform modeling and data analysis for space-based detectors.
- `Numerical methods`: numerical linear algebra, ODE/PDE methods, spectral methods, fitting and diagnostics.

## Status Tags

- `candidate`: 自动抓取或粗筛候选。
- `selected`: 人工或半人工选入核心清单。
- `reading`: 正在阅读。
- `reviewed`: 已有人类阅读笔记。
- `reproducing`: 正在复现。
- `reproduced`: 已复现核心结果。
- `blocked`: 复现或理解受阻，需要记录原因。
- `deprecated`: 保留记录，但不再作为当前主线。

## Human Check

- `no`: 尚未人工校验。
- `partial`: 已检查题目、摘要或部分公式。
- `yes`: 已人工阅读并确认其技术角色。
- `needs expert check`: 需要更高强度的领域校验。
