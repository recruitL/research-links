# 仓库愿景

`research-links` 的目标不是做一个链接收藏夹，而是建设一个长期维护的个人研究方向知识库。

本仓库服务于以下方向：

- EOB / Effective-One-Body theory
- GSF / gravitational self-force
- BHPT / black-hole perturbation theory
- QNM / quasinormal modes
- PM/EFT / post-Minkowskian and effective field theory
- LISA / waveform modeling / data analysis
- numerical methods

核心原则：

- CSV/YAML/JSON 是数据层，保存全量候选记录和可脚本处理字段。
- Markdown 是分析层和展示层，用于解释技术脉络、人工判断和开放问题。
- AI 评价只能做初筛，不能替代人工学术判断。
- 复现状态必须显式记录，不能把阅读印象当作已复现结果。

长期目标是让每个研究方向都能回答三类问题：

1. 这个方向的技术链条如何发展？
2. 哪些论文、公式、代码和数据最值得投入时间？
3. 当前工作可以在哪些位置插入、验证或扩展？
