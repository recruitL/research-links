# Numerical Methods

Numerical methods 目录用于沉淀复现和代码实验所需的数值方法，包括 ODE/PDE、谱方法、线性代数、拟合、误差分析和收敛诊断。

## Core Questions

- 哪些数值方法适合复现 EOB/GSF/BHPT/QNM 中的核心公式？
- 如何记录 convergence diagnostics 和 spurious mode checks？
- 哪些 Python/Julia/C/C++ 工具可以复用？
