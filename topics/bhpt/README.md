# BHPT（Black-Hole Perturbation Theory）

BHPT 研究黑洞背景上的微扰方程，是 GSF、EMRI、QNM 和 test-mass waveform modeling 的基础工具链。

## Core Questions

- 如何在 Schwarzschild/Kerr 背景上描述小扰动？
- Teukolsky、Regge-Wheeler、Zerilli、Sasaki-Nakamura 形式主义之间如何转换？
- BHPT 如何进入 GSF 和 EOB-Teukolsky flux？
