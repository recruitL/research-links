# PM/EFT

PM/EFT 方向关注 post-Minkowskian scattering、effective field theory 和 two-body dynamics，对现代 EOB Hamiltonian 构造和 gauge-invariant diagnostics 有直接影响。

## Core Questions

- PM scattering angle 如何进入 EOB radial potential 或 effective Hamiltonian？
- EFT 计算如何与 PN、GSF 和 NR 信息匹配？
- 当前 PM-informed Hamiltonian diagnostic 能说明什么，不能说明什么？
