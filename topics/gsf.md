# GSF (Gravitational Self-Force)

GSF 方向关注小质量物体在弯曲时空中的自力和辐射反作用，是极端质量比系统、黑洞微扰和高精度两体动力学的重要组成部分。

## Reading Path

1. 从 MiSaTaQuWa 方程和 Quinn-Wald 公理化推导入手，理解自力方程的物理来源。
2. 阅读 point-particle motion 和 self-force 综述，建立奇异场/正则场、mode-sum、puncture 等基本语言。
3. 继续按应用展开：Schwarzschild/Kerr 轨道、自旋/偏心、二阶自力、EMRI 波形、与 EOB/PN/PM 的匹配。
