# QNM（Quasinormal Modes）

QNM 描述扰动黑洞的 ringdown 谱，是 merger-ringdown attachment、黑洞光谱学和稳定性分析的重要组成部分。

## Core Questions

- 如何稳定求解 QNM 频率和本征函数？
- shooting、Leaver continued fraction、WKB、spectral/pseudospectral 和 time-domain extraction 各有什么适用范围？
- 如何识别 spurious modes 和数值收敛问题？
- QNM 谱在非自伴算子意义下什么时候稳定，什么时候需要用 pseudospectrum、EP 或 branch tracking 语言描述？
