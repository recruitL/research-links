# GSF Method Map

## MiSaTaQuWa / Quinn-Wald

基础自力方程，描述小质量物体在自身扰动引起的 regular field 中运动。

## Detweiler-Whiting Decomposition

将场分解为 singular field 和 regular field。regular field 产生物理自力。

## Mode-Sum Regularization

按球谐或 spheroidal modes 分解场，对每个 mode 求解后减去 regularization parameters。

## Puncture / Effective Source

构造局部 puncture 近似奇异场，使剩余场可数值求解。

## Gauge-Invariant Diagnostics

- redshift invariant；
- periastron advance；
- spin-precession invariant；
- tidal invariants；
- ISCO shift。

## Second-Order GSF

二阶 GSF 对 LISA EMRI waveform 精度和 EOB 高精度约束很重要，但技术复杂度高。
