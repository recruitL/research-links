# GSF Problem Map

## Test-Particle Approximation Is Not Enough

EMRI 中小天体不能永远视为无反作用试验粒子；长时间相位精度需要考虑自力和辐射反作用。

## Singular Source Problem

点粒子源导致场在粒子世界线附近奇异，需要 regularization。

## Gauge Problem

自力不是简单的 gauge-invariant 向量。实际比较通常依赖 redshift、periastron advance、spin-precession invariant 等规范不变量。

## Long-Time Evolution

LISA EMRI waveform 需要长时间相位精度，一阶自力、二阶自力和 post-adiabatic expansion 都很关键。

## Connection to EOB

GSF 在小质量比强场区域给出 EOB potentials 和 flux 的约束，是 EOB 与 BHPT/EMRI 的桥梁。
