# Reviews

本目录保存人工论文审查、代码库审查和 AI 初评校验记录。

- `papers/`: paper-level human review.
- `codebases/`: installation, API and theory-transparency notes.
- `ai-evaluations/`: AI-generated evaluations plus human verification.

AI review must not be treated as final scholarship. Every AI note needs a human check status.
