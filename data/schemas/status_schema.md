# Status Schema

## Reading Status

- `candidate`: found by search or import.
- `selected`: selected as likely important.
- `reading`: currently being read.
- `reviewed`: human note exists.
- `revisit`: should be revisited after prerequisite reading.

## Reproduction Status

- `not_started`
- `candidate`
- `in_progress`
- `reproduced`
- `partially_reproduced`
- `blocked`
- `not_reproducible_from_available_info`

## AI/Human Status

- `ai_unreviewed`
- `ai_reviewed`
- `human_unchecked`
- `human_partial`
- `human_checked`
- `needs_expert_check`
